# WhatsApp Web UI Clone

> A whatsapp web ui clone built using **Nextjs** and **styled-components**

### Tech
- Next.js
- styled-components
- styled-normalize
- React Context API
- storybook
- next-themes
- next-images
- svgr
- **Prettier** use with **ESlint** and **Airbnb** code style

### Installation
```
yarn install
```

### Whatsapp Web UI for development
```
yarn dev
```
### Storybook for development
```
yarn storybook
```
